# R-training
Training on R, focused on tidyverse, initially aimed at fellow PhDs in CREAF.
The materials in folder "03 Model selection with information theory (theory, practice, alternatives)", concerns a course that was taught at CREAF (Watering Talents programme, 2025) and through Transmitting Science. The script demonstrates functions using a .txt file called "dataset.drivers.niche.breadth.2026-02-13.txt" that can be downloaded from Zenodo: https://doi.org/10.5281/zenodo.13343361
